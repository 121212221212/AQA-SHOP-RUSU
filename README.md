# Дипломный проект: Автоматизация тестирования сервиса "Путешествие дня"

##  Содержание
- Технологии и инструменты
- Реализованные проверки
- Запуск сайта
- Генерация отчетов
- Файлы проекта

##  Технологии и инструменты
- Java 11
- JUnit 4
- Selenide
- Gradle
- Allure Report
- Lombok

##  Реализованные проверки
- Покупка тура по дебетовой карте
- Покупка тура в кредит по данным карты
- Проверка поддержки карт (одобренные/неодобренные)
- Валидация полей формы


## Запуск сайта
1. Запустить SUT:
   
  - java -jar aqa-shop/aqa-shop.jar
   

##    Генерация отчетов

**Allure Report (Основной отчёт)**

**Генерация статического отчёта**
- ./gradlew allureReport

**Просмотр отчёта**
- **Открыть в браузере:** build/allure-report/index.html
