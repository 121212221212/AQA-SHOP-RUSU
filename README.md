
# Автоматизация тестирования веб-сервиса покупки тура

## Описание проекта

Проект представляет собой автоматизированные тесты для веб-сервиса покупки тура, который взаимодействует с банковскими сервисами и СУБД.

**Функциональность:**
- Оплата тура по дебетовой карте
- Выдача кредита на тур по данным банковской карты

## Предварительные требования

### Обязательные:
- **Java 11** или выше
- **Google Chrome** браузер
- **Gradle** 8.0 или выше

### Рекомендуемые (для полной функциональности):
- **Docker** и **Docker Compose** (для работы с БД)

## Запуск приложения

### 1. Запуск базы данных (опционально):
```bash
docker-compose up -d
```

### 2. Запуск тестируемого приложения:
```bash
java -jar aqa-shop.jar
```

Приложение будет доступно по адресу: http://localhost:8080

## Запуск тестов

### Запуск всех тестов:
```bash
./gradlew test
```

### Запуск конкретных тестов:
```bash
# Только тесты дебетовой карты
./gradlew test --tests "ru.netology.test.DebitCardTest"

# Только тесты кредитной карты  
./gradlew test --tests "ru.netology.test.CreditCardTest"
```

## Генерация отчетов

### Генерация Allure отчета:
```bash
./gradlew allureServe
```

Отчет будет доступен в браузере по адресу: http://localhost:port

## Примечания

- Для работы с базой данных требуется Docker
- Тесты используют предопределенные тестовые карты:
    - APPROVED: 1111 2222 3333 4444
    - DECLINED: 5555 6666 7777 8888
